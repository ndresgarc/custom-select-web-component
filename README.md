# wc-custom-select
A custom select component using Web Components. Mostly because I want to style the options inside it.

## Usage 
```
<custom-select>
    <div value="1">One</div>
    <div value="2">Two</div>
    <div value="3">Three</div>
    <div value="4">Four</div>
<custom-select>
```

## Test
```
npm run dev
```

Visit http://localhost:9001/test/index.html
